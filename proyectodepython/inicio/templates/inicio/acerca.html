Acerca de este blog

Este proyecto es un blog desarrollado con Django como entrega final. Permite registrar usuarios, iniciar sesion, crear y eliminar posteos, visualizar publicaciones, editar el perfil de usuario y utilizar un formulario de contacto.

El objetivo del proyecto es aplicar conceptos basicos de desarrollo web con Python, Django, base de datos, autenticacion, templates y formularios.
